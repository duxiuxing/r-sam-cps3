# CPS3 游戏摘要


## 1. JOJO的奇妙冒险1

《JOJO的奇妙冒险》是日本漫画家荒木飞吕彦的漫画作品。1998 年，卡普空推出同名格斗街机游戏《JOJO的奇妙冒险》。游戏改编自原作漫画第三卷《星尘斗士》，首次将漫画中的“替身”概念引入格斗游戏。玩家可以随时召唤或解散替身，切换本体与替身的战斗模式，开创了独特的双线作战玩法，超必杀 KO 对手时还会以漫画形式表现出来。


## 2. JOJO的奇妙冒险2 未来遗产

卡普空于 1999 年推出的格斗街机游戏《JOJO的奇妙冒险 未来遗产》，是前作《JOJO的奇妙冒险》的“威力加强版”。它在保留核心玩法的基础上，增加了挑战模式以及新角色，并大幅调整了角色平衡性。游戏画面更加精细，动作流畅度极高，完美还原了漫画中各个角色经典的“JOJO立”姿势。


## 3. 赤色大地
《赤色大地》是卡普空于 1996 年推出的格斗街机游戏，也是 CPS-3 基板的首发作品。它最大的特色是“格斗 + RPG”的混血玩法：游戏没有杂兵战，玩家直接挑战八个巨型 BOSS，通过战斗获取经验值升级，解锁新技能，甚至可以用密码续关继承等级。背景设定在中世纪魔幻世界，四位主角（半狮人、功夫少女、忍者、魔女）对抗邪恶巫师瓦尔多尔，画面华丽且系统独特，被誉为格斗游戏中的“异色杰作”。


## 4. 街头霸王3.1

本作是《街头霸王3》系列的首部作品，于 1997 年发售。它最大的特点是角色阵容的“大换血”，除了隆和肯外，其余角色全部为新面孔。游戏引入了革命性的“格挡”系统，允许玩家通过前推摇杆来化解对手的攻击，这一机制彻底改变了格斗游戏的攻防节奏。


## 5. 街头霸王3.2

本作在《新纪元》的基础上增加了五名新角色，包括隐藏角色豪鬼和大巨人雨果。同时，游戏调整了部分角色的平衡性，并增加了新的超必杀技和“EX必杀技”系统，使战斗更具深度。


## 6. 街头霸王3.3

本作是《街头霸王3》系列的的最终版，也是公认的系列巅峰。这次新增加的五名角色中最引人注目的便是初代女性格斗家春丽的回归。春丽能力有了大幅度提高，造型也更加华丽，是游戏中最大的亮点。游戏对平衡性进行了终极优化，并引入了“红色招架”等高级技巧，其流畅的手感、华丽的画面和极高的竞技性使其成为格斗游戏史上的一座丰碑。
